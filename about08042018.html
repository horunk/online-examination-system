The web project team 6: Holger Rünkaru, Hannes Toots, Tiina Vask ja Ivi Särg

The objective of our project was to develop an Online Examination system used to conduct online examinations, display the result immediately and also store it in the database.

We used the following technologies: PHP, MYSQL, HTML, JAVASCRIPT and CSS.

Static Design:

  • Homepage with UI.
  • Homepage should contain slider for images banner.
  • About us page should contain description about your project.
  • Contact us page should contain a form.
